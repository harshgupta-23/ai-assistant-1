5A. APP LAUNCH — NEVER USE EXE NAME ALONE
────────────────────────────────────────────────────────

    NOTEPAD:    r"C:\Windows\System32\notepad.exe"

────────────────────────────────────────────────────────
5B. WINDOW DETECTION — ALWAYS USE PROCESS-BASED MATCHING
────────────────────────────────────────────────────────
  gw.getWindowsWithTitle('Word') is WRONG. Matches any window with 'Word' in title.
  ALWAYS detect windows by process name using psutil + Win32 PID:

    import psutil, ctypes

    def get_windows_by_process(process_name, title_filter=None):
        result = []
        for win in gw.getAllWindows():
            if not win.title.strip():
                continue
            try:
                pid = ctypes.c_ulong()
                ctypes.windll.user32.GetWindowThreadProcessId(
                    win._hWnd, ctypes.byref(pid)
                )
                proc = psutil.Process(pid.value)
                if proc.name().upper() == process_name.upper():
                    if title_filter is None or title_filter in win.title:
                        result.append(win)
            except Exception:
                continue
        return result

  Usage:
    word_wins   = get_windows_by_process('WINWORD.EXE')
    chrome_wins = get_windows_by_process('chrome.exe')
    excel_wins  = get_windows_by_process('EXCEL.EXE')
    vscode_wins = get_windows_by_process('Code.exe')

  EVERY script that interacts with a window must use this function.
  NEVER use gw.getWindowsWithTitle() for app detection.
  gw.getWindowsWithTitle() is ONLY acceptable for browser tab title checks.

────────────────────────────────────────────────────────
5C. WINDOW FOCUS — ALWAYS USE WIN32 WITH VERIFICATION
────────────────────────────────────────────────────────
  pygetwindow.activate() silently fails on most Windows setups.
  NEVER use it as the primary focus method.

  ALWAYS use this verified force-focus function:

    def force_focus_verified(win, retries=3):
        for attempt in range(retries):
            try:
                ctypes.windll.user32.ShowWindow(win._hWnd, 9)
                ctypes.windll.user32.SetForegroundWindow(win._hWnd)
                time.sleep(0.5)
                fg = ctypes.windll.user32.GetForegroundWindow()
                if fg == win._hWnd:
                    return True
            except Exception as e:
                print(f"[ERROR] Focus attempt {attempt+1} failed: {e}")
        return False

  Only fall back to win.activate() if force_focus_verified() returns False.
  ALWAYS add time.sleep(0.5) after any focus attempt.
